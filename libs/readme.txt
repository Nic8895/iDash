rF2 Shared Memory Plugin by The Iron Wolf

Version 1.1.0.1 - 3/10/2017

Setup Instructions:

Copy rFactor2SharedMemoryMapPlugin64.dll to [rF2]\Bin64\Plugins\ folder.

If you are running rFactor 2 in 32bit mode (not a common scenario), copy rFactor2SharedMemoryMapPlugin32.dll to [rF2]\Bin32\Plugins\ folder.


For more info, see: https://forum.studio-397.com/index.php?threads/rf2-shared-memory-plugin.54282/#post-875258
and http://thecrewchief.org/showthread.php?2-rFactor-2-Crew-Chief-Setup-Instructions.
